# Live Wallpaper for macOS

A professional, high-performance Live Wallpaper application for macOS.

## Features
- **Live Video Wallpapers**: Supports 4K video playback.
- **Menu Bar App**: Runs quietly in the background with no Dock icon.
- **Smart Power Management**: Automatically pauses when unplugged, sleeping, or locked.
- **Wallpaper Library**: Manage your collection with persistent thumbnails.
- **Launch at Login**: Start automatically when you sign in.

## Installation
1. Download the app.
2. Move to Applications folder.
3. Right-click and Open (to bypass Gatekeeper).

## Requirements
- macOS 13.0 or later.
